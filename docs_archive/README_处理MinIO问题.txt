========================================
yanying MinIO 502问题 - 简明处理指南
========================================

您看到的"minio not initialized"和"502 Bad Gateway"错误
是MinIO Go SDK与MinIO服务器之间的兼容性问题。

----------------------------------------
推荐方案：切换到本地存储（最简单）
----------------------------------------

执行以下命令：

cd /code/EasyDarwin/build/EasyDarwin-lin-v8.3.3-202510161428/configs
sed -i "s/store = 'minio'/store = 'local'/" config.toml
cd ..
pkill -9 easydarwin  
./easydarwin &

这样您就可以立即使用所有功能，图片保存到本地。
